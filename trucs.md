# Per provar els canvis fets a la web abans de penjar-los a la web real:
1r obro la terminal i em poso a la carpeta de la web.
2n escric la comanda "npm install" i seguidament després "npm run dev"
3r després ja estaria em sortirà "Local: http://localhost:5000/" aquell link és el que serveix per visualitzar-ho abans de penjar a la web oficial

http://localhost:5173/
